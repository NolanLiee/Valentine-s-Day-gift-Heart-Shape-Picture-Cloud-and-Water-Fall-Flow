# Valentine’s Day gift：Heart-Shape-Picture-Cloud and Water-Fall-Flow of Pictures
Use local pictures to generate a lightweight and cutie heart shape picture cloud and waterfall flow of pictures to cheer up your gf/bf.

1. index is the main function of this generator based on javascript
2. images is the pictures you want to use to generate the cloud and the flow, fomat: 001.jpg ... 999.jpg
3. pichuli.py is a Image Batch Processing Script based on magika, which enables you to modify the size and name of your raw pictures.
